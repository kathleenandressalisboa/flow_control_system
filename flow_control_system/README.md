## 📌 Project Goal
SnackBar Order Flow is a simple Python project created to practice programming concepts and track the evolution of a system step by step.

## 🚀 Phase 1 – Basic CLI
In this first phase, the system works through a Command Line Interface (CLI).

Features implemented:
- Add a new order
- List all orders
- Show the total value of orders
- Interactive menu in the terminal

## 🛠 Technologies
- Python 3

## ▶ How to Run

Clone the repository:

git clone https://github.com/kathleenandressalisboa/snackbar-order-flow.git

Run the program:

python main.py

## 📚 Learning Goals
This phase focuses on practicing:
- Python functions
- Lists and tuples
- User input handling
- Basic CLI menu structure
